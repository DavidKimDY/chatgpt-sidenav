# ChatGPT 질문 사이드바 익스텐션

ChatGPT 대화에서 질문들을 사이드바로 확인하고, 클릭하여 해당 질문으로 부드럽게 스크롤할 수 있는 크롬 익스텐션입니다.

## 주요 기능

- 📋 ChatGPT 대화의 모든 질문을 사이드바에 자동으로 표시
- 🎯 질문 클릭 시 해당 질문으로 부드럽게 스크롤
- 🔄 실시간 업데이트 (새 질문이 추가되면 자동 반영)
- 📱 접기/펼치기 기능으로 화면 공간 활용
- ✨ 깔끔한 UI 디자인

## 설치 방법

1. 크롬 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

## 사용 방법

1. ChatGPT 웹사이트(https://chat.openai.com)에 접속
2. 대화를 시작하면 화면 오른쪽에 사이드바가 자동으로 나타납니다
3. 사이드바에서 질문 목록을 확인할 수 있습니다
4. 질문을 클릭하면 해당 질문으로 부드럽게 스크롤됩니다
5. 사이드바 헤더의 `−` 버튼을 클릭하면 접을 수 있습니다

## 파일 구조

- `manifest.json` - 익스텐션 설정 파일 (content script, 권한 등)
- `content.js` - ChatGPT 페이지에 주입되는 스크립트 (질문 추출, 사이드바 생성)
- `sidebar.css` - 사이드바 스타일시트
- `popup.html` - 익스텐션 팝업 HTML
- `popup.js` - 팝업 JavaScript
- `styles.css` - 팝업 스타일시트

## 작동 원리

1. **Content Script**: ChatGPT 페이지(`chat.openai.com`)에 자동으로 주입됩니다
2. **질문 추출**: 페이지의 DOM을 분석하여 사용자가 작성한 질문들을 찾습니다
3. **사이드바 생성**: 화면 오른쪽에 고정된 사이드바를 생성합니다
4. **실시간 업데이트**: MutationObserver를 사용하여 새 메시지가 추가될 때마다 자동으로 업데이트합니다
5. **스크롤 기능**: 질문 클릭 시 `scrollIntoView`를 사용하여 부드럽게 스크롤합니다
